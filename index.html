Histoire

Android, ses outils.

Le langage de programmation Java a été choisi par Google pour créer les applications Android. Outre Java, le métalangage XML est utilisé pour la conception d'interfaces. Pour créer des applications, vous pouvez utiliser le kit de développement d'Android : Android Studio.

Le kit est un ensemble d'outils qui inclut un débogueur et des bibliothèques logicielles. Avec Android, les changements sont fréquents. Chaque année, Google lance une nouvelle version de son système opérationnel. L'avant-dernière, Oreo, et la dernière Popcorn, gagnent de plus en plus d'utilisateurs. Les versions plus anciennes perdent progressivement du terrain. Cependant, pour un débutant, l'idée de devoir apprendre tout un langage pour créer une simple application pour être décourageante. C'est pourquoi nous vous conseillons de vous lancer dans la programmation en faisant usage de ces quelques outils de développement. Ils vous permettront de créer une application Android en toute simplicité, juste avec des notions basiques en programmation. Certes, vous n’obtiendrez pas de résultats bien professionnels, mais ce sera déjà un bon début.
